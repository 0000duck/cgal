\ccDefinition

\ccc{CGAL::write_xyz_point_set()} saves points (positions + optionally normals) to a .xyz file (ASCII).

The \ccc{CGAL::write_xyz_point_set()} function exists in two flavors: 
the input can be either a container of \ccc{Point_3} or of \ccc{PointWithNormal_3} points.
