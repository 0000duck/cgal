\ccDefinition  
The class \ccRefName\ is a model of the concept
\ccc{RegularTriangulationFaceBase_2}. It is the default face base class
of regular triangulations.

\ccInclude{CGAL/Regular_triangulation_face_base_2.h}


\ccParameters
The template parameters \ccc{Traits} has to be a model
of \ccc{RegularTriangulationTraits_2}.

The template parameter \ccc{Fb} has to be a model
of \ccc{TriangulationFaceBase_2}. By default, this parameter is
instantiated by
\ccc{CGAL::Triangulation_face_base_2<Traits>}.
