\begin{ccRefClass}{HalfedgeDS_ccb_base<Refs>}

\ccDefinition
  
The class \ccRefName\ is a model of the \ccc{HalfedgeDSCcb}
concept. \ccc{Refs} is an instantiation of a \ccc{HalfedgeDS}. 
The full declaration states four template parameters:

\begin{tabbing}
\mbox{}~~~~~~ \= \ccc{template <} \=\ccc{class Refs,}\\
              \> \>\ccc{class Tag_face} \=\ccc{= CGAL::Tag_true>}\\
     \> \ccc{class HalfedgeDS_ccb_base;}
\end{tabbing}

If \ccc{Tag_face} $\equiv$ \ccc{CGAL::Tag_true} a bounded face is
supported.

In all cases, a reference to the primary halfedge is supported.
