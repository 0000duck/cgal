\begin{ccRefClass}{HalfedgeDS_ccb_min_base<Refs>}

\ccDefinition
  
The class \ccRefName\ is a model of the \ccc{HalfedgeDSCcb}
concept. \ccc{Refs} is an instantiation of a \ccc{HalfedgeDS}. 
It is equivalent to \ccc{CGAL::HalfedgeDS_ccb_base< Refs, 
CGAL::Tag_false>}.
The class contains support for the face pointer and
the required type definitions. It can be used for deriving own ccbs.
