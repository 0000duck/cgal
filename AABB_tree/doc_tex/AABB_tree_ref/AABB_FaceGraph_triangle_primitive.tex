\begin{ccRefClass}{AABB_FaceGraph_triangle_primitive<FaceGraph,cache_datum>}

\ccDefinition
  
The class \ccRefName\ is a model of the concept \ccc{AABBPrimitive}. It wraps a facet handle of a polyhedron, which is used as id.
The polyhedron from which the primitive is built should not be deleted while the AABB tree holding the primitive is in use.

\ccParameters
The template parameter \ccc{FaceGraph} must be an instantiation of the class \ccc{CGAL::Polyhedron_3<Traits>}.
Using \ccc{CGAL::Kernel_traits<T>}, the kernel \ccc{K} of the point type of the Polyhedron is used
to define the nested types \ccc{Point} and \ccc{Datum} of the primitive class, being \ccc{K::Point_3} and \ccc{K::Triangle_3} respectively.\\
\ccc{cache_datum} is either \ccc{CGAL::Tag_true} or \ccc{CGAL::Tag_false}. In the former case, the datum is stored in the primitive, while in the latter it is constructed on the fly to reduce
the memory footprint. The default is \ccc{CGAL::Tag_false} (datum is not stored).
