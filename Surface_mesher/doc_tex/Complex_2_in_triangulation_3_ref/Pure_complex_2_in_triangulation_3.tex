\ccDefinition
  
The class \ccRefName\  implements a data structure to store
a two dimensional pure  complex 
embedded in a three dimensional triangulation.
The template parameter \ccc{Tr} has to be instantiated
with a model of the concept \ccc{Triangulation_3}.
(Any three dimensional   triangulation of
\cgal\ is a model of \ccc{Triangulation_3}.)

The class \ccRefName\  can be plugged as template parameter
in the global function \ccc{make_surface_mesh<C2T3,Oracle,Criteria,Tag>}.



\ccInclude{Pure_complex_2_in_triangulation_3.h}

\ccIsModel

\ccc{PureComplex_2InTriangulation_3}

WARNING :
Currently the class  \ccRefName\  does not implement the
whole concept \ccc{PureComplex_2InTriangulation_3}
but just what is needed to  be used as the first  parameter 
of  \ccc{make_surface_mesh<C2T3,Oracle,Criteria,Tag>}.
