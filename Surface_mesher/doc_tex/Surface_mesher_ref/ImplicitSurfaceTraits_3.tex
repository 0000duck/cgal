\ccDefinition
  
The concept \ccRefName\ provides a numerical type, \ccc{FT},
and a point type, \ccc{Point_3}.

The class \ccc{Implicit_surface_3< Traits, Func>}
is a model to be plugged 
as  second template parameter of the surface mesher
\ccc{make_surface_mesh<C2T3,SurfaceOracle,Criteria,Tag>}. 
Whenever a model of this traits is plugged
in \ccc{Implicit_surface_3< Traits, Func>} 
to provide a type for  implicit surfaces to be
meshed, 
the types \ccc{FT} and \ccc{Point_3}
provided are required to match respectively  the numerical type
and the point type of the triangulation type \ccc{C2t3::Triangulation_3}
used to represent the three dimensional triangulation embedding
the two dimensional meshed surface.
