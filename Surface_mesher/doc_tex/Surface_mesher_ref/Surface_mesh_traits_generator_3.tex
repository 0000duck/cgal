\ccDefinition
  
The class \ccRefName\ provides 
 a type \ccc{Type}, that is 
a model of the concept \ccc{SurfaceMeshTraits_3}
for the surface type \ccc{Surface}.

The type \ccc{Surface} is required to be a model of the concept
\ccc{Surface_3},
which means that it is copy constructible
and assignable.
 In addition, a \ccc{Surface} type is required
\begin{itemize}
\item 
either to provide a nested type
\ccc{Surface::Surface_mesher_traits_3}
that is a model of \ccc{SurfaceMeshTraits_3}
\item or to be a surface type for which a specialization
of the traits generator \ccRefName\ exists.
\end{itemize}
 

Currently, the library provides partial specializations
of  the   traits generator 
for implicit surfaces (\ccc{Implicit_surface_3<Traits, Function>}) and 
gray level images (\ccc{Gray_level_image_3<FT, Point>}).
