\ccDefinition
  
The class \ccRefName\ provides 
provides a type \ccc{Type} , that is 
a model of the concept \ccc{SurfaceMeshTraits_3}
for the surface type \ccc{Surface}.

The mecanism used here is similar to the
\ccc{Kernel_traits mecanism}. 
More precisely, the type \ccc{Surface} is required to 
either provide a nested type
\ccc{Surface::Surface_mesher_traits_3}
that is a model of \ccc{SurfaceMeshTraits_3}
or to be a surface type for which a specialization
of the traits generator \ccRefName exists.
 

Currently, the library provides partial specializations
of  the   traits generator 
for implicit surfaces (\ccc{Implicit_surface_3<Traits, Func> }) and 
grey level images (\ccc{Gray_level_image_3<Traits> }).
