\begin{ccRefEnum}[Periodic_3_triangulation_3::]{Locate_type}

\ccDefinition
  
The enum \ccRefName\ is defined by \ccc{Periodic_3_triangulation_3} to
specify which case occurs when locating a point in the
triangulation. If the triangulation does not contain any points
\ccc{EMPTY} is returned.

\ccEnum{enum Locate_type {VERTEX=0, EDGE, FACET, CELL, EMPTY};}
{}
