\ccDefinition
  
The class \ccRefName\ provides a default triangulation to be used as the 3D
triangulation of a mesh generation process.

\ccInclude{CGAL/Mesh_triangulation_3.h}

\ccParameters
Template parameter \ccc{MD} stands for a model of \ccc{MeshDomain_3}. 

Template parameter \ccc{Gt} stands for a model of \ccc{RegularTriangulationTraits_3} 
and defaults to \ccc{Kernel_traits<MD>::Kernel}.

% \ccc{Gt} defaults to MD::

% It is a 3D regular
% triangulation whose vertex and cell classes are respectively
% \ccc{Mesh_vertex_base_3<MD,Gt,Vb>} and
% \ccc{Mesh_cell_base_3<MD,Gt,Cb>}.
