\ccDefinition
  
The class \ccRefName\ is a model of the concept \ccc{MeshCellBase_3}.
It is designed to serve as cell base class for the 3D triangulation 
used in the 3D mesh generation process.

\ccParameters

The  template parameter \ccc{MD} provides the types of indices
used to identify 
the faces of the input complex. It has to be a model
of the concept \ccc{MeshDomain_3}.

The template parameter \ccc{Gt} is the geometric traits class.
It  has to be a model of the concept \ccc{RegularTriangulationTraits_3}.

The third parameter \ccc{Cb} is the cell base class. It has to be a model
of the concept \ccc{RegularTriangulationCellBase_3} and defaults to
\ccc{Regular_triangulation_cell_base_3<Gt>}.
