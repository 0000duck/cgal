\ccDefinition
  
The class \ccRefName\ is a model of \ccc{MeshCriteria_3}. 
It gathers the criteria for  facets in the boundary surface meshes
and for mesh tetrahedra.

\ccInclude{CGAL/Mesh_criteria_3.h}

\ccParameters
The parameter \ccc{Tr} must be identical to the nested type
\ccc{Triangulation} of the instance used as model of
\ccc{MeshComplex_3InTriangulation_3}.
