\ccDefinition

The class \ccRefName{} is a model for the concept \ccc{TriangleAccessor_3}. It is
designed to serve as accessor for objects of type \ccc{CGAL::Polyhedron_3<K>}.

\begin{ccAdvanced}
Actually, the class \ccRefName{} is a partial specialization of the class
template \ccc{template<typename Polyhedron, typename K>
  Triangle_accessor_3<Polyhedron, K>}. One may give another partial
specialization of this class to handle one's own polyhedron data structure.
\end{ccAdvanced}

\ccInclude{CGAL/Triangle_accessor_3.h}

\ccParameters

The template parameter \ccc{K} is the geometric traits class.
