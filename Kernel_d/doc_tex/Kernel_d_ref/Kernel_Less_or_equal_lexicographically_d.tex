\begin{ccRefFunctionObjectConcept}{Kernel::Less_or_equal_lexicographically_d}
A model for this must provide:

\ccCreationVariable{fo}

\ccMemberFunction{bool operator()(const Kernel::Point_d& p, const
  Kernel::Point_d& q);} {returns \ccc{true} iff $p$ is
  lexicographically smaller than $q$ with respect to Cartesian
  lexicographic order of points or equal to $q$. \ccPrecond{\ccc{p}
  and \ccc{q} have the same dimension.}}

\end{ccRefFunctionObjectConcept}
