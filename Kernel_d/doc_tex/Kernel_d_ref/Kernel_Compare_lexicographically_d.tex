\begin{ccRefFunctionObjectConcept}{Kernel::Compare_lexicographically_d}
A model for this must provide:

\ccCreationVariable{fo}

\ccMemberFunction{Comparison_result operator()(const Kernel::Point_d&
  p, const Kernel::Point_d& q);}{Compares the Cartesian coordinates of
  points \ccStyle{p} and \ccStyle{q} lexicographically in ascending
  order of its Cartesian components \ccc{p[i]} and \ccc{q[i]} for $i =
  0,\ldots,d-1$. \ccPrecond{The objects are of the same dimension.}}

\end{ccRefFunctionObjectConcept}
