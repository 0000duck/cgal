\ccDefinition
A model for \ccc{Kernel} that derives from \ccc{Base} (already a
\ccc{Kernel}) and redefines the predicates as filtered predicates
using \ccc{AK} as the approximate kernel and \ccc{EK} as the exact
kernel.
