\ccDefinition
  
The concept \ccRefName\ describes a 3d polyhedral surface inheriting
from the class \ccc{CGAL::Polyhedron_3}, whose faces are all triangular.

\ccHasModels

\ccc{CGAL::Polyhedron_3} with the restriction that faces are triangular.
