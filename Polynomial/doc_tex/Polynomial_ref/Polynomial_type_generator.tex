\begin{ccRefClass} {Polynomial_type_generator<T,d>} \label{Polynomial_type_generator}

\ccDefinition

This class template provides a convenient way to obtain the type representing a multivariate polynomial with \ccc{d} variables, where \ccc{T} is the innermost coefficient type. In case \ccc{T} happens to be a \ccc{CGAL::Polynomial} the generator will add \ccc{d} variables to \ccc{T}.\\

\ccc{T} must be a model of \ccc{IntegralDomainWithoutDivision}.\\
\ccc{d} must be of type int. 

\ccTypes

\ccNestedType{Type}{The generated type.}
