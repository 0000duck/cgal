\begin{ccRefConcept}{AlgebraicKernel_d_2::Solve_2}

\ccDefinition

\ccCreationVariable{fo}

A model \ccVar\ of this type must provide:

\ccMethod{template < class OutputIterator >
    OutputIterator
    operator()(const AlgebraicKernel_d_2::Polynomial_2 & p1,
	     const AlgebraicKernel_d_2::Polynomial_2 & p2,
	     OutputIterator res);}
{Copies in the output iterator the common roots of $p_1$ 
and $p_2$ with their multiplicity, as objects of type
\ccc{std::pair< AlgebraicKernel_d_2::Algebraic_real_2, int >}.
\ccPrecond{$p_1$ and $p_2$ are square-free and the set of solutions of 
the system is 0-dimensional (i.e., the two polynomials must be
coprime).}}
