\ccDefinition
  
The class \ccRefName\ is designed as a default traits class for the
class \ccc{Periodic_3_triangulation_3<Periodic_3TriangulationTraits_3,TriangulationDataStructure_3>}. 

The argument \ccc{Traits} must be a model of the
\ccc{DelaunayTriangulationTraits_3} concept. The 
argument \ccc{Periodic_3Offset_3} must be a model of the concept
\ccc{Periodic_3Offset_3} and defaults to \ccc{Periodic_3_offset_3}.

Note that this template class is specialized for
\ccc{CGAL::Filtered_kernel}, so that it automatically provides
filtered predicates. This holds implicitly for
\ccc{CGAL::Exact_predicates_inexact_constructions_kernel}, as it is an
instantiation of \ccc{CGAL::Filtered_kernel}.
