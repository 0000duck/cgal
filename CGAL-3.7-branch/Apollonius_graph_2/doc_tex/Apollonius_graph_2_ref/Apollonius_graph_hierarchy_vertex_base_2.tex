\ccDefinition

The class \ccRefName\ provides a model for the
\ccc{ApolloniusGraphHierarchyVertexBase_2} concept, which is the
vertex base required by the
\ccc{Apollonius_graph_hierarchy_2<Gt,Agds>} class. The class
\ccRefName\ is templated by a class \ccc{Agvb} which must be a model
of the \ccc{ApolloniusGraphVertexBase_2} concept.
