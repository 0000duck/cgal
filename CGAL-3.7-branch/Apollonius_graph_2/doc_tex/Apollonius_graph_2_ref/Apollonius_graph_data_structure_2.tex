\ccDefinition

An implementation for the concept \ccc{ApolloniusGraphDataStructure_2}
is provided. The class is templated by a face 
base and vertex base class just like the
\ccc{Triangulation_data_structure_2} class. The \ccRefName\ class
inherits from \ccc{Triangulation_data_structure_2<Vb,Fb>}. \ccc{Vb}
must be a model of \ccc{ApolloniusGraphVertexBase_2}. \ccc{Fb}
must be a model of \ccc{TriangulationFaceBase_2}.
