\ccDefinition
  
The class \ccRefName\ is a model of the concept \ccc{MeshVertexBase_3}.
It is designed to serve as vertex base class for the 3D triangulation 
used in a 3D mesh generation process.

\ccParameters

The template parameter \ccc{MD}  provides  the types of indices
used to identify 
the faces of the input complex. It must be a model
of the concept \ccc{MeshDomain_3}.

The template parameter \ccc{Gt} is the geometric traits class.
It  must be a model of the concept \ccc{RegularTriangulationTraits_3}.

The third parameter \ccc{Vb} is the vertex base class. It has to be a model
of the concept \ccc{TriangulationVertexBase_3} and defaults to
\ccc{Triangulation_vertex_base_3<Gt>}.
