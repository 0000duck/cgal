\begin{ccRefConcept}{SkinSurfaceQuadraticSurfaceTraits_3}
  \ccDefinition

  The concept \ccRefName\ defines the traits class used for the
  \ccc{Skin_surface_quadratic_surface_3}. It defines the geometric
  objects (ring type, points, vectors and segments) for the quadratic
  surface together with a few geometric predicates and constructions
  on these objects.

  \ccThree{SkinSurfaceQuadraticSurfaceTraits_3}{Segment_3}{}
  \ccThreeToTwo

  \ccTypes

  \ccNestedType{RT}{The ring type.}\ccGlue
  \ccNestedType{Point_3}{The point type.}\ccGlue
  \ccNestedType{Vector_3}{The vector type.}\ccGlue
  \ccNestedType{Segment_3}{The segment type.}
