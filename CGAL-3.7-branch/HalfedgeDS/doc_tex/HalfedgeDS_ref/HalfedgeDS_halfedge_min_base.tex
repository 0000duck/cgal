\begin{ccRefClass}{HalfedgeDS_halfedge_min_base<Refs>}

\ccDefinition
  
The class \ccRefName\ is a model of the \ccc{HalfedgeDSHalfedge}
concept. \ccc{Refs} is an instantiation of a \ccc{HalfedgeDS}. 
It is equivalent to \ccc{CGAL::HalfedgeDS_halfedge_base< Refs, 
CGAL::Tag_false, CGAL::Tag_false, CGAL::Tag_false>}.
The class contains support for the next and the opposite pointer and
the required type definitions. It can be used for deriving own halfedges.
