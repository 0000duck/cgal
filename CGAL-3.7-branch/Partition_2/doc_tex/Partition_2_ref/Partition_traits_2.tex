\ccDefinition
  
Traits class that can be used with all the
2-dimensional polygon partitioning algorithms.  It is parameterized by
a representation class \ccc{R}.

({\em i.e.}, \ccc{Less_yx_2}, \ccc{Less_xy_2}, \ccc{Left_turn_2},
