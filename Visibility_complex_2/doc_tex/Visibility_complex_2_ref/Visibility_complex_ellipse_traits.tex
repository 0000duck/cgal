\begin{ccRefClass}{Visibility_complex_2_ellipse_traits<Kernel>}

\ccDefinition

Geometric traits class to compute a visibility complex of ellipses, from
type \ccc{Conic_2<Kernel>}. The template parameter is a kernel. This traits
class does not provide distance functions, and can therefore not be used
with \ccc{Shortest_path_2}.

Using a bigint type such as \ccc{Gmpz} is recommended.
