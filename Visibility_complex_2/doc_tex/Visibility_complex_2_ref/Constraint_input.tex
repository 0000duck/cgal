\begin{ccRefClass}{Visibility_complex_2_details::Constraint_input}

\ccDefinition

The class \ccRefName{} is a utility to represent constraints
passed as input to \ccc{Visibility_complex_2<VC2GeomTraits,VC2Items>} and
\ccc{Compute_free_bitangents<VC2GeomTraits,VC2Items>}. It assumes the disks
are passed in a separate sequence, and represents a constraint by its type,
and the index in the sequence of disks of the source and target object.

This class is defined inside the private subnamespace
\ccc{CGAL::Visibility_complex_2_details}, but made accessible through
\ccc{Visibility_complex_2<VC2GeomTraits,VC2Items>} and
\ccc{Compute_free_bitangents<VC2GeomTraits,VC2Items>}.

\ccTypes
\ccNestedType{Type}{The geometric type of bitangents. This is the same type
as the one nested in the bitangents.}
