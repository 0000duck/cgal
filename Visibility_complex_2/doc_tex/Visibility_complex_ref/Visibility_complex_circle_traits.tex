\begin{ccRefClass}{Visibility_complex_2_circle_traits<Kernel>}

\ccDefinition

Geometric traits class to compute a visibility complex of circles with a
rational radius. The template parameter is a kernel. This traits class does
not provide distance functions, and can therefore not be used with
\ccc{Shortest_path_2}.

Using a bigint type such as \ccc{Gmpz} is recommended.
