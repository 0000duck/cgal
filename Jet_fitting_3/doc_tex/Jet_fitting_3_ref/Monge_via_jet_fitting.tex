\ccDefinition
  
The class \ccRefName\ is designed to perform the estimation of the
local differential quantities at a single point. This point is the
first point amongst the input points given by two iterators on
points. According to the specified degrees for the polynomial fitting
and for the Monge representation, it outputs the results through two
classes \ccc{Monge_rep} and \ccc{Monge_info}.
 
\ccInclude{Monge_via_jet_fitting.h}

\ccParameters
The class \ccRefName\ has three template parameters. The first parameter
\ccc{DataKernel} shall provide basic geometry for input points and
some \ccc{Monge_rep} members. The second parameter \ccc{LocalKernel}
shall provide basic geometry for local computation. The third
parameter \ccc{LinAlgTraits} provides the linear algebra computation
tools.

\ccConstructor{Monge_via_jet_fitting(Range_Iterator begin, Range_Iterator end, 
int d, int dprime, Monge_rep &monge_rep, Monge_info &monge_info);}
{This constructor performs all the computations. The $N$ input points
are given by the \ccc{Range_Iterators}, \ccc{d} is the degree of the
fitted polynomial, \ccc{dprime} is the degree of the expected Monge
coefficients, outputs are stored in \ccc{monge_rep} and
\ccc{monge_info}.
\ccPrecond $N \geq N_d=(d+1)(d+2)/2$, $1 \leq d$, $dprime \leq d$, $1
\leq dprime \leq 4$ }
