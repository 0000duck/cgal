\ccDefinition
  
The concept \ccRefName\ describes the set of requirements to be
fulfilled by any class used to instantiate the second template
parameter of the class
\ccc{Monge_via_jet_fitting<DataKernel,LocalKernel,SvdTraits>}.
 
This concept provides the geometric primitives used for the
computations in the class
\ccc{Monge_via_jet_fitting}. 

\ccRequirements
% +------------------------------------------------------------------
 
In the class \ccc{Monge_via_jet_fitting} the scalar type,
\ccc{LocalKernel::FT}, must be the same as that of the \ccc{SvdTraits}
concept~: \ccc{SvdTraits::FT}.

% The concept \ccRefName\ must fulfill the requirement of
% the\ccc{Kernel} template of the
% class\ccc{CGAL::Aff_transformation_3<Kernel>}.

% The type \ccc{LocalKernel::FT} must fulfill the requirement of
% the\ccc{FT} template of the class\ccc{CGAL::Linear_algebraCd<FT>}.

 The type \ccc{LocalKernel::FT} is a model of the FieldWithSqrt concept.

% The type \ccc{LocalKernel::FT} must fulfill the requirement of the
% \ccc{FT} template of the function \ccc{CGAL::Sqrt<NT>}. An alternative 
% is to overload this latter function.

\ccOperations
% +------------------------------------------------------------------
The scalar type \ccc{LocalKernel::FT} must be a field type with a
square root.

Only constructors (from 3 scalars and copy constructors) and access
methods to coordinates x(), y(), z() are needed for the point and
vector types.
