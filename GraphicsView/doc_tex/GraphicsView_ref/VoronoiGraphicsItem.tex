
\begin{ccRefClass}{VoronoiGraphicsItem<DT>}

\ccDefinition An object of type \ccRefName\ is a graphics item that
encapsulates a Delaunay triangulation in order to draw its dual, the
Voronoi diagram.

\ccInclude{CGAL/IO/VoronoiGraphicsItem.h}

\ccParameters

The template parameter \ccc{DT} of \ccc{VoronoiGraphicsItem<DT>} must be a \cgal\ Delaunay triangulation class.


\ccInheritsFrom
\ccc{QtGraphicsItem}

\ccCreation
\ccCreationVariable{vgi}

\ccConstructor{VoronoiGraphicsItem<DT>(DT* dt);}{Constucts a graphics item for the dual of the 
Delaunay triangulation \ccc{dt}.}

\ccMethod{QPen edgesPen()() const;}{Get the pen that is used to draw edges.}
\ccMethod{void setEdgesPen()(const QPen& p);}{Set the pen that is used to draw edges.}

\ccMethod{bool drawEdges();}{.}
\ccMethod{bool setDrawEdges(bool b);}{.}
\end{ccRefClass}
