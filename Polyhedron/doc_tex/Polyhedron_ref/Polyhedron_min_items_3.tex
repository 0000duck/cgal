\begin{ccRefClass}{Polyhedron_min_items_3}

\ccDefinition
  
The class \ccRefName\ is a minimal model of the
\ccc{PolyhedronItems_3} concept.  It provides definitions for vertices
containing points, halfedges, and faces. The polyhedron traits class
must provide the respective type for the point. Vertices and facets
both do \emph{not\/} contain a halfedge handle to an incident
halfedge.
