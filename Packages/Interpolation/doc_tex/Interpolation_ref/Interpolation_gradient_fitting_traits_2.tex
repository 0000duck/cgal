  \ccDefinition \ccRefName\ is a model for the concepts
  \ccc{InterpolationTraits} and \ccc{GradientFittingTraits}. It can be
  used to instantiate the geometric traits class of interpolation
  functions and of Sibson's gradient fitting function when applied on
  a bivariate function over a two-dimensional domain. The traits class
  is templated by a kernel class \ccc{K}.
