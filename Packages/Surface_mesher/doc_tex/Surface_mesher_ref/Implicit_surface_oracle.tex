\ccDefinition
  
The class \ccRefName\  implements a model of the concept
\ccc{SurfaceOracle} corresponding to the case
of an implicit surface. 

Assume that the  surface is
the zero level set of the function
$f : \R^3 \longrightarrow R$. The template parameter
\ccc{Func}  is to be instantiated by a model of the concept
\ccc{ImplicitFunction} whose \ccc{operator()} computes the values 
of the function $f$.
