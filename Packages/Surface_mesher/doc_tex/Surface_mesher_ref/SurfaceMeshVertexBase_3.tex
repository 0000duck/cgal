\ccDefinition
  
The \ccc{Surface_mesh_complex_2_in_triangulation_3<Tr>} data structure is able to store
a pure two dimensional complex embedded in a triangulation.
The concept \ccRefName\ describes the base vertex type to be
used  in the  triangulation class \ccc{Tr}  plugged in a
\ccc{Surface_mesh__complex_2_in_triangulation_3<Tr>} data structure.



\ccGeneralizes

\ccc{TriangulationVertexBase_3}

The concept \ccRefName\ 
add functions to store and retrieve the status of the vertex.
This feature is required to allow a caching mecanism
for the vertex status.
%a boolean \ccc{visited} marker 
%and a caching mechanism to store the 
%answer of an \ccc{is_regular()} query.
