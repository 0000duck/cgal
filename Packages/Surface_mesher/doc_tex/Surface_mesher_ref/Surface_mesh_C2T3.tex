\ccDefinition
  
The class \ccRefName\  implements a data structure to store
a two dimensional pure  complex 
embedded in a three dimensional triangulation.
The template parameter \ccc{Tr} has to be instanciated
with a model of the concept \ccc{Triangulation_3}.
(Any three dimensional   triangulation of
\cgal\ is a model of \ccc{Triangulation_3}.)

The class \ccRefName\  can be plugged as template parameter
in the global function \ccc{surface_mesher<C2T3,Oracle,Criteria,Tag>}.



\ccInclude{Surface_mesh_C2T3.h}

\ccIsModel

\ccc{SurfaceMeshC2T3}

WARNING :
Currently the class  \ccRefName\  does not implement the
whole concept \ccc{SurfaceMeshC2T3}
but can stll be used as the first  parameter class
of  \ccc{surface_mesher<C2T3,Oracle,Criteria,Tag>}.
