\ccDefinition
%============
The traits class \ccRefName\ models the concept 
\ccc{GeneralPolygonSetTraits_2}. It enables Boolean set-operations on 
(linear) polygons. It defines the exposed type \ccc{Polygon_2} to be 
\ccc{CGAL::Polygon_2<Kernel,Container>}. By default, the template 
parameter \ccc{Container} is instantiated by 
\ccc{std::vector<Kernel::Point_2>} and the template parameter
\ccc{ArrSegmentTraits} is instantiated by 
\ccc{CGAL::Arr_segment_traits_2<Kernel>}.
