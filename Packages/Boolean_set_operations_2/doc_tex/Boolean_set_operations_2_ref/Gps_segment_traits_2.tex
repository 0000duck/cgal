\ccDefinition
%============
The traits class \ccRefName\ models the \ccc{GeneralPolygonSetTraits_2}
concept. It enables Boolean set-operations on (linear) polygons.
It should be parameterized with a kernel, which is used in turn to
define the exposed type \ccc{General_polygon_2} to be of type
\ccc{Polygon_2<Kernel>}.
