\ccDefinition

This operator imports a polygon or a polygon with holes from the input
stream \ccc{in}. Only the point coordinates of the boundary are used.
If the stream \ccc{in} does not contain a permissible polygon or a
polygon with holes, the \ccc{ios::badbit} of the input stream \ccc{in}
is set and $P$ remains unchanged.

An ASCII and a binary format exist. The stream detects the
format automatically and can read both.
