\ccDefinition
%============
The traits class \ccRefName\ models the concept 
\ccc{GeneralPolygonSetTraits_2}. It inherits from the instantiated 
type of the template parameter \ccc{ArrTraits}, which must model the 
concept \ccc{ArrangementDirectionalXMonotoneTraits}, (which in turn refines
the concept \ccc{ArrangementXMonotoneTraits}). The template parameter
\ccc{GeneralPolygon} must be instantiated with a model of the concept
of \ccc{GeneralPolygon_2}. By default, the latter is instantiated by
\ccc{CGAL::General_polygon_2<ArrTraits>}.
