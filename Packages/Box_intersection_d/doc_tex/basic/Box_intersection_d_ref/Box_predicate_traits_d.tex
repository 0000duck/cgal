\begin{ccRefClass}{Box_predicate_traits_d<BoxTraits_d, bool closed>}
\ccDefinition
Default implementation of the \ccc{BoxPredicateTraits_d} concept. First template parameter is a \ccc{Box_traits_d} model. Second parameter specifies the topology, which can be half-open or closed. In the half-open case, two adjacent boxes do not intersect, whereas in the closed case they do.
