\begin{ccRefClass}{Box_traits_d<Box>}

\ccDefinition
There are two template specializations of \ccc{Box_traits_d<>}: one for pointer
to boxes, one for immediate box types. The template parameter must be a box that
is similar to the default box called \ccc{Box_d}.
