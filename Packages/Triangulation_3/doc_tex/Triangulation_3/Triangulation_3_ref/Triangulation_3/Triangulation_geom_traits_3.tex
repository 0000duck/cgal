\ccDefinition
  
The class \ccRefName\ is the predefined geometric traits class
provided by \cgal. It uses the kernel geometric objects and
predicates.  This class is templated with a representation class.

The traits class \ccRefName\ is designed to deal with \cgal\ three
dimensional points. It can be used as a default traits
class for \ccc{Triangulation_3<Triangulation_traits_3,Tds_3>},
\ccc{Delaunay_triangulation_3<Triangulation_traits_3,Tds_3>} and
\ccc{Delaunay_hierarchic_triangulation_3<Triangulation_traits_3,Tds_3>}.
