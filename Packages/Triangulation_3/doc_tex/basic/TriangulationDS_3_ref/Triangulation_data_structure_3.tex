\ccDefinition
The class \ccRefName\ stores a 3D-triangulation data structure
and provides the optional
geometric functionalities to be used as a parameter for a 
3D-geometric triangulation (see Chapter~\ref{chapter-Triangulation3}). 

It is parameterized by base classes for vertices and cells, which have the
default values \ccc{Triangulation_ds_vertex_base_3<>} and
\ccc{Triangulation_ds_cell_base_3<>} respectively.
\lcTex{(See \ccRefPage{TriangulationDSVertexBase_3} 
and \ccRefPage{TriangulationDSCellBase_3}.)}

\ccIsModel

\ccc{TriangulationDataStructure_3}

\ccInheritsFrom{\ccc{Triangulation_utils_3}}

This class \ccc{Triangulation_utils_3} defines basic computations on
indices of vertices and neighbors of cells.
