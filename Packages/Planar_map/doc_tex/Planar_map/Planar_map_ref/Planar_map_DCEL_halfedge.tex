\begin{ccRefConcept}[Planar_map_DCEL::]{Halfedge}
\ccRefLabel{Pm_Dcel_Halfedge}
\ccCreationVariable{h}

The \ccRefName concept is a refinement of the \ccc{Topological_map_DCEL_Halfedge} concept. 
The requirements of the latter are the requirements of the former.
In addition the \ccRefName should contain the  following geometric types and methods.

\ccTypes
\ccNestedType{Curve}{a curve associated with the halfedge.}

  \ccOperations
  \ccMethod{const Curve& curve();}{the curve.}
  
  \ccMethod{void set_curve(const Curve& cv);}{set the curve in \ccVar{} to $cv$.}

\end{ccRefConcept} % Planar_map_DCEL::Halfedge
