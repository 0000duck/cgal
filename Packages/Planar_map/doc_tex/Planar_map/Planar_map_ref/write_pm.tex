%write_pm
\begin{ccRefFunction}{write_pm<Planar_map,Writer>(Planar_map pm, Writer& writer, ostream& o);}

\ccCreationVariable{write_pm}

\ccDefinition
\ccc{write_pm} is a global function defined for writing a \ccc{Planar map} object. 
The function is templated with the \ccc{Planar map} class, and the \ccStyle{Writer} class.  
This function uses the functions defined in the \ccStyle{Writer} class, and prints a  
\ccc{Planar map} object in the format discussed in \ccRefPage{Pm_IO_format}. 
The operator defined for writing a \ccc{Planar map} obejct to an output stream 
uses this function when parameterize \ccStyle{Writer} with an object instanciated from \ccc{Pm_file_writer}.

\ccInclude{CGAL/IO/write_pm.h}

\end{ccRefFunction} % writer_pm
