\begin{ccRefClass}{Arr_face_graph_adator<PlanarMap>}
    
\ccDefinition
%============

The class uses as a adaptor from \ccc{Arrangement_2} to a BOOST graph. It 
defines various 
types and methods needed for a graph to adapted into boost. \ccc{Arrangement_2} 
faces are considered as the verticies of the graph. More particulary, 
\ccc{Arrangement_2::Face_handle} are the graph verticies and 
\ccc{Arrangement_2::Halfedge_handle} are the edges of the graph.

The arrangement template has one parameter:
\begin{itemize}
\item The \ccc{PlanarMap} template-parameter should be instantiated with
a class that is a model of the \ccc{ArranagementDcel} concept. This parameter 
is the planar map to be adapted.
\end{itemize}

The adaptor implements three of BOOST graph concepts:
\begin{itemize}
\item Graph.
\item Incidence Graph.
\item Vertex List Graph.
\end{itemize}
