\begin{ccRefClass}{Polyhedron_default_traits_3<R>}

\ccDefinition
  
The class \ccRefName\ is a model of the \ccc{Polyhedron_traits}
concept.  It defines the geometric types and primitive operations
used in the polyhedral surface data structure
\ccc{CGAL::Polyhedron_3<Traits>} in terms of the \cgal\ kernel, which is 
given through its kernel representation class \ccc{R}.

\ccInclude{CGAL/Polyhedron_default_traits_3.h}

\ccIsModel

\ccc{Polyhedron_traits}

\ccTypes
\ccThree{typedef CGAL::Vector_3<R>;}{Normal;}{}

\ccTypedef{typedef CGAL::Point_3<R> Point;}{}
\ccGlue
\ccTypedef{typedef CGAL::Plane_3<R> Plane;}{}
\ccGlue
\ccTypedef{typedef CGAL::Vector_3<R> Normal;}{}

\ccCreation
\ccCreationVariable{traits}  %% choose variable name
\ccTwo{CGAL::Polyhedron_default_traits_3<R> traits;;;}{}

\ccConstructor{Polyhedron_default_traits_3();}{default constructor.}

\ccExample

Instantiation of a polyhedral surface with the default traits and a 
Cartesian kernel based on double coordinates. 

\ccIncludeExampleCode{Polyhedron/polyhedron_prog_simple.C}

\end{ccRefClass}
