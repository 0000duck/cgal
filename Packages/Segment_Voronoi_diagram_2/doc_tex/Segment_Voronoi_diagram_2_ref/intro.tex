\chapter{2D Segment Voronoi diagrams}
\label{ref-chapter-apollonius2}

\ccHeading{Definition}

\cgal\ provides the class
\ccc{CGAL::Segment_Voronoi_diagram_2<Gt,DS>} for
computing the 2D Voronoi diagram of segments and points. The two
template parameters must be models of the
\ccc{SegmentVoronoiDiagramTraits_2} and
\ccc{ApolloniusGraphDataStructure_2} concepts. The first concept
is related to the geometric objects and predicates associated with
segment Voronoi diagrams, whereas the second concept refers to the data
structure used to represent the dual of the segment Voronoi
diagram. The classes \ccc{Segment_Voronoi_diagram_traits_2<K,MTag>},
\ccc{Segment_Voronoi_diagram_traits_without_intersections_2<K,MTag>}
\ccc{Segment_Voronoi_diagram_filtered_traits_2<K,MTag>}
\ccc{Segment_Voronoi_diagram_filtered_traits_without_intersections_2<K,MTag>}
are models of the \ccc{SegmentVoronoiDiagramTraits_2} concept, whereas
the class
\ccc{Apollonius_graph_data_structure_2<Vb,Fb>} is a model of the
\ccc{ApolloniusGraphDataStructure_2} concept.
