\ccDefinition

An implementation for the concept \ccc{SegmentVoronoiDiagramDataStructure_2}
is provided. The class is templated by a face 
base and vertex base class just like the
\ccc{Triangulation_data_structure_2} and
\ccc{Apollonius_graph_data_structure_2} class. This class
inherits from \ccc{Apollonius_graph_data_structure_2<Vb,Fb>}. \ccc{Vb}
must be a model of \ccc{SegmentVoronoiDiagramVertexBase_2}. \ccc{Fb}
must be a model of \ccc{TriangulationFaceBase_2}.
