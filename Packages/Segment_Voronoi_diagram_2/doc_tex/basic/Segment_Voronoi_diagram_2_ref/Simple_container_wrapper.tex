\begin{ccRefClass}{Simple_container_wrapper<C>}

%% \ccHtmlCrossLink{}     %% add further rules for cross referencing links
%% \ccHtmlIndexC[concept]{} %% add further index entries
\ccCreationVariable{vb}
\ccDefinition

The class \ccRefName\ describes a wrapper for realizing a model of the
\ccc{SimpleContainer} concept. The template parameter \ccc{C} must be
a model of the \ccc{ReversibleContainer} and \ccc{BackInsertionSequence}
concepts of STL. Moreover, it is required that \ccc{C}'s iterators are
not invalidated when insertions and deletions are performed in \ccc{C}.
A model for \ccc{C} is \ccc{template<typename T> std::list<T>}.

\ccIsModel
\ccc{SimpleContainer}



\ccTypes
\ccTwo{SimpleContainer::value_type+}{}
The class \ccRefName\ defines the following type, in addition to the
types required by the \ccc{SimpleContainer} concept:

\ccTypedef{typedef C Container;}
{A type for the template parameter \ccc{C}.}
