\ccDefinition

The class \ccRefName\ implements an input iterator
for the computation of approximate nearest neighbours.

\ccInclude{CGAL/Orthogonal_priority_search.h}

\ccParameters

Expects for the parameter \ccc{Traits} an implementation of the concept TreeTraits.
Expects for the parameter \ccc{QueryItem}  an implementation of a $d$-dimensional point, 
for example \ccc{CGAL::Point_d}.

Expects for the parameter \ccc{Distance} an implementation of the concept OrthogonalDistance. 

\ccCreation
\ccCreationVariable{i}  %% choose variable name

\ccConstructor{iterator();}{Default constructor. Constructs an past-the-end iterator.}
\ccConstructor{iterator(iterator& Iter);}{Copy constructor.}
\ccConstructor{iterator(Tree& tree, QueryItem& q, Traits tr, Distance& d, int k, NT Eps,
bool Search_nearest=true);}
{Creates an iterator for searching approximately neighbours of the query item \ccc{q}
in the items stored in the $k$-$d$ tree \ccc{tree}.}
