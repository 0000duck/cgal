\begin{ccRefFunction}{marching_tetrahedra_3}

\ccDefinition

The function \ccRefName\ implements the marching tetrahedra algorithm
\cite{cgal:tpg-rmtiise-99}. 

The input is either an input range or a triangulation. The value type
of the input iterator is \ccc{Cell_handle_3}. The four vertices of a
cell are indexed with 0, 1, 2 and 3 in positive orientation, the
positive orientation being defined by the orientation of the
underlying Euclidean space $\R^3$, which is the case for
\ccc{Triangulation_3}, cf.\ Section~\ref{Triangulation3-sec-intro}.
% NGHK: check link above...

The output is a \ccc{Polyhedron_3} and is constructed using the
\ccc{Polyhedral_incremental_builder_3}.
