\section{Provided Kinetic Data Structures}
\label{sec:provided_kdss}

Here we present a using a kinetic data structure--Delaunay
triangulations in two dimensions--. The framework provides some
additional kinetic data structures, for example, sorting on x
coordinate (\ccc{CGAL::KDS::Sort<Traits>}), Delaunay triangulation in
3 dimensions (\ccc{CGAL::KDS::Delaunay_triangulation_3<Traits,
Visitor, Triangulation>} and regular triangulations in 3D
(\ccc{CGAL::KDS::Regular_triangulation_3<Traits, Visitor,
Triangulation}) as well as bounding boxes to make visual simulations
more appealing (\ccc{CGAL::KDS::Enclosing_box_2<Traits>},
\ccc{CGAL::KDS::Enclosing_box_3<Traits>}. These are documented in the
reference manual.
