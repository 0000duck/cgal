\begin{ccRefClass} {Straight_skeleton_vertex_2<HdsVertex>}

\ccDefinition

The class \ccClassTemplateName\ represents a vertex of the straight skeleton.

A straight skeleton contains two kinds of vertices: border and inner. A border vertex is a vertex of the input polygon. An inner vertex is a vertex of the skeleton itself (the joint of three bisectors).

Every vertex in the straight skeleton has degree 3 or higher (including border vertices).} 

\ccInheritsFrom

\ccc{HdsVertex}

This class is the underlying vertex class.
