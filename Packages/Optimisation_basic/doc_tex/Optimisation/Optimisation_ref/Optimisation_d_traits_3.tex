\ccDefinition

The class \ccRefName\ is a traits class for \ccc{Min_annulus_d<Traits>}
using the two-dimensional \cgal\ kernel.
 
The first template argument is expected to be a \cgal\ representation class.
The second and third template argument have default type \ccc{R::RT}.
