\begin{ccRefFunctionObjectConcept}{ConstructDualPoint_2}

The functor \ccc{ConstructDualPoint_2} is responsible for constructing
the point dual to a (triangular) face of the Delaunay graph. This
point is the Voronoi vertex of the three sites defining the face in
the Delaunay graph. The result type of the functor is the point type
defined in the Voronoi traits providing this functor.

\ccRefines
\ccc{DefaultConstructible}, \ccc{CopyConstructible}, \ccc{Assignable},
\ccc{AdaptableFunctor} (with one argument)

\ccTypes

In addition to the types required by the \ccc{AdaptableFunctor}
concept, a model of this concept must provide the following types:

\ccTwo{ConstructDualPoint_2::Face_handle+}{}
\ccNestedType{Face_handle}{A type for handles of the faces in the
  Delaunay graph.}

\ccCreationVariable{cdp}

\ccOperations
%It must also provide the following operations:

\ccThree{result_type}{cdp(Face_handle f)+}{}
\ccThreeToTwo
\ccMemberFunction{result_type operator()(Face_handle f);}
{Returns the dual point of the face in the Delaunay graph
  corresponding to the face handle \ccc{f}. The face handle \ccc{f}
  cannot correspond to an infinite face.}
