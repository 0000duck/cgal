\begin{ccRefFunctionObjectConcept}{AccessSite_2}

The functor \ccc{AccessSite_2} is responsible for accessing the site
corresponding to the vertex handle passed to it as an argument. The
result type of this functor is the site type of the Voronoi traits
providing this functor, or a const reference to this site type.

\ccRefines
\ccc{DefaultConstructible}, \ccc{CopyConstructible}, \ccc{Assignable},
\ccc{AdaptableFunctor} (with one argument)

\ccTypes

In addition to the types required by the \ccc{AdaptableFunctor}
concept, a model of this concept must provide the following types:

\ccTwo{SiteAccess_2::Vertex_handle+}{}
\ccNestedType{Vertex_handle}{A type for handles of the vertices in the
  Delaunay graph.}

\ccCreationVariable{as}

\ccOperations
%It must also provide the following operations:

\ccThree{result_type}{as(Vertex_handle v)+}{}
\ccThreeToTwo
\ccMemberFunction{result_type operator()(Vertex_handle v);}
{Returns the site corresponding to the vertex handle \ccc{v} in the
  Delaunay graph. The vertex handle \ccc{v} cannot correspond to the
  infinite vertex.}
