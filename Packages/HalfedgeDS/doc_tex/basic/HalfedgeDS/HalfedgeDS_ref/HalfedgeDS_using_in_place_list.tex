\begin{ccRefClass}{HalfedgeDS_using_in_place_list<Traits,HalfedgeDSItems>}

\ccDefinition
  
The class \ccRefName\ is a model for the \ccc{HalfedgeDS} concept.
\ccRefName\ is a list-based representation with bidirectional
iterators that supports removal.

\ccInclude{CGAL/HalfedgeDS_using_in_place_list.h}

\ccImplementation

\ccRefName\ uses internally the \ccc{CGAL::In_place_list} container class.
gThe copy constructor and the assignment operator are implemented
sub-optimally in $O(n \log n)$ for the list representation with $n$ the
number of vertices plus halfedges plus faces.
