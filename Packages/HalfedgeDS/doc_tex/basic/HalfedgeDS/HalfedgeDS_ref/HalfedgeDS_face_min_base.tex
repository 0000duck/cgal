\begin{ccRefClass}{HalfedgeDS_face_min_base<Refs>}
  
The class \ccRefName\ is a model of the \ccc{HalfedgeDS_Face}
concept. \ccc{Refs} is an instantiation of a \ccc{HalfedgeDS}.  It is
equivalent to \ccc{CGAL::HalfedgeDS_face_base< Refs, CGAL::Tag_false>}. 
It is empty besides the required type definitions. It can be used for 
deriving own faces. If the compiler does not support partial specialization, 
i.e., macro \ccc{CGAL_CFG_NO_PARTIAL_CLASS_TEMPLATE_SPECIALISATION} is set, 
this class is preferable to \ccc{CGAL::HalfedgeDS_face_base< Refs, 
CGAL::Tag_false>}.
