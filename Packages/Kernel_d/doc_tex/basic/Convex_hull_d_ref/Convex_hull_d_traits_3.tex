\ccc{Convex_hull_d}.  This is a traits class that adapts any
low-dimensional standard kernel model, e.g. \ccc{Homogeneous<RT>} or
\ccc{Cartesian<FT>} for the fixed 3-dimensional usage of
\ccc{Convex_hull_d}.
