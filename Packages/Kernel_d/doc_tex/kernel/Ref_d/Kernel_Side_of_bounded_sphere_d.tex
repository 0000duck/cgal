\begin{ccRefFunctionObjectConcept}{Kernel::Side_of_bounded_sphere_d}
A model for this must provide:

\ccCreationVariable{fo}

\ccMemberFunction{template <class ForwardIterator>
Bounded_side operator()(
  ForwardIterator first, ForwardIterator last, const Kernel::Point_d& p);}
{returns the relative position of point \ccc{p} to the sphere
  defined by \ccc{A = tuple [first,last)}. The order of the points of
  $A$ does not matter.  
  \ccPrecond value type of \ccc{ForwardIterator} is \ccc{Kernel::Point_d} and
  \ccc{orientation(first,last)} is not \ccc{ZERO}.}

\end{ccRefFunctionObjectConcept}
