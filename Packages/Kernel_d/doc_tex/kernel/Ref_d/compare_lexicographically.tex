compare_lexicographically(const Point_d<R>& p,
                          const Point_d<R>& q)}
{Compares the Cartesian coordinates of points \ccStyle{p} and
 \ccStyle{q} lexicographically in order of its Cartesian 
 components \ccc{p.cartesian[i]} and \ccc{q.cartesian[i]}
 for $i = 0,\ldots,d-1$.}
