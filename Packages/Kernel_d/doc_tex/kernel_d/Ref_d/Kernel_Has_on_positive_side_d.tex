\begin{ccRefFunctionObjectConcept}{Kernel::Has_on_positive_side_d}
A model for this must provide:

\ccCreationVariable{fo}

\ccMemberFunction{template <class Kernel_object>
bool operator()(const Kernel_object& o, const Kernel::Point_d& p);}
{returns true iff $p$ is on the positive side of $o$. 
\ccc{Kernel_object} may be any of \ccc{Kernel::Sphere_d}, 
\ccc{Kernel::Hyperplane_d}.}

\end{ccRefFunctionObjectConcept}
