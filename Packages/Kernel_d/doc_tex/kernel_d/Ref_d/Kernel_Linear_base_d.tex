\begin{ccRefFunctionObjectConcept}{Kernel::Linear_base_d}
A model for this must provide:

\ccCreationVariable{fo} \ccMemberFunction{template <class
  ForwardIterator, class OutputIterator> int
  operator()(ForwardIterator first, ForwardIterator last,
  OutputIterator result); }{computes a basis of the linear space
  spanned by the vectors in \ccc{A = tuple [first,last)} and returns
  it via an iterator range starting in \ccc{result}. The returned
  iterator marks the end of the output.  \ccPrecond $A$ contains
  vectors of the same dimension $d$. \ccRequire The value type of
  \ccc{ForwardIterator} and \ccc{OutputIterator} is
  \ccc{Kernel::Vector_d}.}

\end{ccRefFunctionObjectConcept}
