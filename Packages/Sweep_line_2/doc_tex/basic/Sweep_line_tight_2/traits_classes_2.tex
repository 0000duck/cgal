\section{Traits Classes}
The \ccStyle{Sweep_line_2} class 
is parameterized by a traits class, which specifies the types and geometric
primitives to be used in the computation. 
The \ccc{PlanarMapWithIntersectionsTraits_2} concept is a refinement of the 
 \ccc{SweepLineTraits_2} concept.
Hence, all implementations of Planar Map With Intersections Traits 
provided in \cgal are model of the \ccc{SweepLineTraits_2} concept.
