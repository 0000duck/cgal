\begin{ccRefConcept}{FieldNumberType}

A number type representing field elements, i.e., must be closed under
operations $+$, $-$, $*$ and also under division operation $/$.

\ccRefines
\ccc{CopyConstructable}, \ccc{Assignable}, \ccc{RingNumberType}

\ccSeeAlso

\end{ccRefConcept}
