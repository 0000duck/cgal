\begin{ccRefFunction}{operator>>}
\label{refNef_polyhedron_operator_rightshift_3}

\ccDefinition

This operator reads a Nef polyhedron, which is given in the proprietary file
format written by the output operator. Finite Nef polyhedra can be loaded
with both, the standard and the extended kernel. Infinite Nef polyhedra can
only be loaded with the extended kernel.
