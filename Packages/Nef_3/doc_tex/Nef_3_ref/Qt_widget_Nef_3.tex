\begin{ccRefClass}{Qt_widget_Nef_3<Nef_polyhedron_3}

\ccDefinition
 
The class \ccc{Qt_widget_Nef_3} uses the OpenGL interface of QT to display a
\ccc{Nef_polyhedron_3}. Its purpose is to provide an easy to use viewer for 
\ccc{Nef_polyhedron_3}. There are no means provided to enhance the 
functionality of the viewer.

In addition to the functions inherited from the Qt class \ccc{OGLWidget} via 
\ccc{Qt_widget_OpenGL}, \ccc{Qt_widget_Nef_3} only has a single public 
constructor. For the usage of \ccc{Qt_widget_Nef_3} see the example 
below and section  in the User Manual.

\ccInclude{CGAL/IO/Qt_widget_Nef_3.h}

\ccParameters
The template parameter expects an instantiation of \ccc{Nef_polyhedron_3<Traits>}.
 
\ccInheritsFrom
\ccc{Qt_widget_OpenGL}

\ccCreation
