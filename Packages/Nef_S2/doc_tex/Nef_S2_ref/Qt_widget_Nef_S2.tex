\begin{ccRefClass}{Qt_widget_Nef_S2<Nef_polyhedron_S2}

\ccDefinition
 
The class \ccc{Qt_widget_Nef_S2} uses the OpenGL interface of QT to display a
\ccc{Nef_polyhedron_S2}. Its purpose is to provide an easy to use viewer for 
\ccc{Nef_polyhedron_S2}. There are no means provided to enhance the 
functionality of the viewer.

In addition to the functions inherited from the Qt class \ccc{OGLWidget} via 
\ccc{Qt_widget_OpenGL}, \ccc{Qt_widget_Nef_S2} only has a single public 
constructor. For the usage of \ccc{Qt_widget_Nef_S2} see the example 
below and section  in the User Manual.

\ccInclude{CGAL/IO/Qt_widget_Nef_S2.h}

\ccParameters
The template parameter expects an instantiation of \ccc{Nef_polyhedron_S2<Traits>}.
 
\ccInheritsFrom
\ccc{Qt_widget_OpenGL}

\ccCreation
\ccCreationVariable{W}

\ccConstructor{Qt_widget_Nef_3(const Nef_polyhedron_S2& N);}
{a widget \ccVar\ for displaying the \ccc{Nef_polyhedron_S2} N.}

This example create some random \ccc{Sphere_segments} and distributes them on
two \ccc{Nef_polyhedron_2}. The two Nef polyhedra are combined by a symmetric
diffrence and the result is displayed in a Qt widget.
