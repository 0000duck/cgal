\begin{ccRefClass}{Fast}

\ccDefinition
\ccRefName\ is a tag class.  It can be used to parameterize a complexity policy
in order to specify a particularly fast variant of an algorithm.
For example, passing \ccc{Location_policy<Fast>} as parameter to
\ccc{Delaunay_triangulation_3} selects a fast point location at the expense of
more memory usage.

\ccInclude{CGAL/Complexity_tags.h}

\ccIsModel
\ccc{DefaultConstructible, CopyConstructible}

\ccSeeAlso
\ccc{Location_policy} \\
\ccc{Compact} \\
\ccc{Compact_location} \\
\ccc{Fast_location}

\end{ccRefClass} 


\begin{ccRefClass}{Compact}

\ccDefinition

For example, passing \ccc{Location_policy<Compact>} as parameter to
\ccc{Delaunay_triangulation_3} selects a slower point location which saves memory.

\begin{ccRefClass}{Location_policy<Tag>}

\ccDefinition
\ccRefName\ is a policy class which can be used to specify a trade-off
between memory usage and time complexity for the point location strategy
used in a data-structure.
For example, passing \ccc{Location_policy<Compact>} as parameter to
\ccc{Delaunay_triangulation_3} selects a slower point location which saves memory.

\ccInclude{CGAL/Location_policy.h}

\ccParameters

\ccc{Tag} can only be either \ccc{Fast} or \ccc{Compact} currently.

\ccIsModel
\ccc{DefaultConstructible, CopyConstructible}

\ccSeeAlso
\ccc{Compact} \\
\ccc{Fast} \\
\ccc{Fast_location} \\
\ccc{Compact_location}

\end{ccRefClass}
