\ccDefinition

The class \ccRefName\ provides a {\sc Bgl} 
\ccAnchor{http://www.boost.org/libs/property_map/ReadablePropertyMap.html}{readable property map} 
for the {\em internal property} \ccc{edge_is_border} 
required by the \ccc{HalfedgeGraph} concept.

It indicates if a \ccc{CGAL::Polyhedron_3} halfedge is a border halfedge.

The template parameter \ccc{Traits} is the same \ccc{Traits} parameter of 
\ccc{CGAL::Polyhedron_3<Traits>}.
