\ccDefinition

The class \ccc{Eigen_sparse_matrix} is a C++ wrapper around Eigen's matrix type \ccc{Eigen::SparseMatrix}
that represents general matrices (symmetric or asymmetric).

\ccDefinition

The class \ccc{Eigen_sparse_symmetric_matrix} is a C++ wrapper around Eigen's matrix type \ccc{Eigen::SparseMatrix}.

This kind of matrix is symmetric, only the lower triangle is stored.
