\ccFunction{void orient_normals_wrt_cameras_3(VertexIterator first, VertexIterator beyond, PointMap const& vertex_point, NormalMap& vertex_normal, CamerasMap const& vertex_cameras);}
{
Orient the normals of the [first, beyond) range of vertices w.r.t. the position of cameras that reconstructed the points by photogrammetry.
Preconditions:\begin{itemize}
\item VertexIterator is a model of ForwardIterator.\item PointMap is a model of \ccc{boost::readable_property_map_tag}.\item NormalMap is a model of \ccc{boost::lvalue_property_map_tag}.\item CamerasMap is a model of \ccc{boost::readable_property_map_tag}. \end{itemize}
}
