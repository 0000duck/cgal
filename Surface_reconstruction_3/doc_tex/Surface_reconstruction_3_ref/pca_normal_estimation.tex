\ccDefinition

\ccc{CGAL::pca_normal_estimation()} estimates normal directions at all points from an input point set by linear least squares fitting of a plane over their k nearest neighbors. The result is an unoriented normal vector for each input point. The function exists in two flavors: it may either require a kernel or deduces it from the input data.
