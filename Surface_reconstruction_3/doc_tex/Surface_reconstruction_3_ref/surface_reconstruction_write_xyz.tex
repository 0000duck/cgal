\ccDefinition

\ccc{CGAL::surface_reconstruction_write_xyz()} saves points (positions + optionally normals) to a .xyz file (ASCII).

The \ccc{CGAL::surface_reconstruction_write_xyz()} function exists in two flavors: 
the input can be either a container of \ccc{Point_3} or of \ccc{PointWithNormal_3} points.
