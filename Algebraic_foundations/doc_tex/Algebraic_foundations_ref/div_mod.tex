\begin{ccRefFunction}{div_mod}

\ccDefinition

The function \ccRefName\ computes the integral quotient and remainder of division with remainder.The function is guranteed to be well defined in case NT 
is a model of the \ccc{EuclideanRing} concept. 

\ccInclude{CGAL/number_utils.h}

\ccFunction{template <class NT> void div_mod(const NT& x, 
                                             const NT& y, 
                                                   NT& q, 
                                                   NT& r);}{ 
        computes the quotient $q$ and remainder $r$, such that $x = q*y + r$ 
        and $r$ minimal with respect to the Euclidean Norm on   
        \ccc{Algebraic_structure}.}
