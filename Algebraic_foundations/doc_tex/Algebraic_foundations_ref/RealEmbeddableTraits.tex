\begin{ccRefConcept}{RealEmbeddableTraits}

\ccDefinition

A model of \ccc{RealEmbeddableTraits} is associated to a number type 
\ccc{Real_embeddable} and reflects the properties of this type with respect 
to the concept RealEmbeddable.

Whether a type is a model of RealEmbeddable or not is indicated by the 
Is\_real\_embeddable\_tag. 

In case \ccc{Real_embeddable} is a model of RealEmbeddable the provided 
functors are not \ccc{CGAL::Null_functor}. 

\ccTypes

A model of \ccc{RealEmbeddableTraits} is supposed to provide:\\

\ccNestedType{Real_embeddable}{The associated number type.}

\ccNestedType{Is_real_embeddable}
        { Tag indicating whether \ccc{Real_embeddable} is real embeddable. \\
          This is either \ccc{CGAL::Tag_true} or \ccc{CGAL::Tag_false}. }
