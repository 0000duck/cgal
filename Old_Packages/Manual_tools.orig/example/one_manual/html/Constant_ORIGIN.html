CGAL::ORIGIN

Definition

The constant ORIGIN does this and that.

#include <ORIGIN.h>

type ORIGIN = init_value;

See Also

Some_other_variable, some_other_function.

Example

A short example program. Instead of a short program fragment, a full running program can be included using the \ccIncludeExampleCode{examples/Example/ORIGIN_prog.C} macro. The program example would be part of the source code distribution and also part of the automatic test suite.
