Spaces Around Pointer * and Reference & Symbols

Reference or pointer parameters can occur on both sides of the separating space between the return type and the function or method name. The formatting normalizes them to the left side. This formatting is not done within the2 parameters (, but maybe in the future). An example:

Demo_Class& foo1 ( int& a, int* b)
Demo_Class* foo2 ( int& a, int* b)
Demo_Class & foo3 ( int &a, int *b)
Demo_Class * foo4 ( int &a, int *b)
