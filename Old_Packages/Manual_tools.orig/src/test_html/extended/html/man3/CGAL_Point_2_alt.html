2D Point B (CGAL_Point_2)

Definition

An object of the class CGAL_Point_2 is a point in the two-dimensional euclidean plane 2.

Creation

#include <CGAL/Point_2.h>

CGAL_Point_2<R> p ( R::RT hx, R::RT hy, R::RT hw = R::RT(1));
introduces a point p initialized to (hx/hw,hy/hw). If the third argument is not explicitely given it defaults to R::RT(1).
