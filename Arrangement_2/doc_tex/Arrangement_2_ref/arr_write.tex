Writes the arrangement \ccc{arr} into the given output stream using a
specific output format. \ccc{formatter}, which must be a model of the
\ccc{ArrangementOutputFormatter}, defines the output format.

\ccGlobalFunction{template<typename Traits, typename Dcel, typename Formatter>

                                 Formatter& formatter);}
